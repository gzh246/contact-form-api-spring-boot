# Security policy

## Supported version

The latest commit on the `main` branch is the only supported version of this
demonstration project.

## Reporting a vulnerability

Please do not disclose a vulnerability, credential, private URL, or customer
data in a public issue.

Use GitHub private vulnerability reporting when it is available for this
repository. If it is not available, open a minimal public issue titled
`Security contact request`. Do not include technical details, affected URLs,
payloads, identifiers, logs, credentials, or personal data. Ask only for a
private reporting channel and wait for the maintainer to respond.

Do not test the public demo with destructive payloads or real personal data.
